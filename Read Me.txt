# n8n Scholarship AI Agent

An AI-powered automation workflow that scrapes scholarship websites, extracts structured data using Gemini AI, and automatically stores results in Google Sheets.

## Features

- Automated scholarship scraping
- AI-based data extraction
- Duplicate detection
- Google Sheets database
- Fully automated workflow using n8n

## Tech Stack

- n8n
- Gemini AI
- Google Sheets API
- Web scraping

## Workflow Overview

1. Scrape scholarship websites
2. Extract scholarship information using AI
3. Convert output into structured JSON
4. Check duplicates in Google Sheets
5. Append new scholarships automatically

## Data Extracted

- Scholarship Title
- Country
- Scholarship Link
- Degree Level
